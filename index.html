Solid Base

A solid foundation, for me, starts with operators, flow control and language orientation (there are some frameworks that allow you to program aspect-oriented).

Base Formats

The base formats in all high-level languages are characters, numbers and time. I think it is important to learn how to handle them to save a lot of time.

Collections & JSON

Collections allow us to manage keyed, structured and indexed data. JavaScript Object Notation, more commonly known by the acronym JSON, is an open data interchange format that is both human and machine-readable.

Memory Manage

Memory management, or rather stack management, is vital to scale applications correctly and avoid double work.

Asyncronous Code

Control abstractions can help to structure code, especially async code (without using deeply nested callbacks, for example).

Interceptable Objects

In javascript the interceptable or proposed instances for this purpose are Reflect and Proxy. They contain static internal methods that allow default forwarding behavior on proxy handler traps.

Internationalization API

The Intl namespace object contains several constructors, as well as functions common to internationalization constructors and other language-sensitive functions. Together, they comprise the ECMAScript Internationalization API, which provides language-sensitive string comparison, number formatting, date and time formatting, and more.
